## labeling-tool

## 개요

이 프로젝트는 웹에서 실행 가능한 labeling-tool 입니다.

## 개발환경

Visual Studio Code
Chrome
JavaScript
CSS
React 16.13.0
Redux 4.0.5
그 외 상세 dependencies는 package.json에서 확인할 수 있습니다.

## 실행 방법

폴더 생성 후 해당 폴더에서 git bash 실행.

### git clone git@github.com:dlog6649/labeling-tool.git
### cd labeling-tool
### npm i
### npm start
