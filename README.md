# Labeling Tool

## Description

This project is a labeling tool that operates in a web browser environment. React was used as the UI library.

## Environment

- vite v4.3.3
- node.js v18.14.0
- pnpm v8.3.1
- react v16.13.0
- visual studio code

## Scripts

### Start development server
```bash
$ git clone git@github.com:dlog6649/labeling-tool.git
$ cd labeling-tool
$ pnpm i
$ pnpm dev
```

### Test

```bash
pnpm test
```

### Build

```bash
pnpm build
```
