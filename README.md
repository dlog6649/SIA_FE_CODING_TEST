# labeling-tool

## 개요

이 프로젝트는 웹 브라우저 환경에서 동작하는 라벨링 툴입니다.
UI 라이브러리로는 React를 사용했습니다.

## 개발환경

- vite v4.3.3
- node.js v18.14.0
- yarn v1.22.19
- react v16.13.0
- visual studio code

## 개발 환경 실행 

```bash
$ git clone git@github.com:dlog6649/labeling-tool.git
$ cd labeling-tool
$ yarn install
$ yarn dev
```

## 테스트

```bash
yarn test
```
