## labeling-tool

## 개요

이 프로젝트는 웹에서 작동하는 labeling-tool 입니다.

## 개발환경

- React 16.13.0
- Redux 4.0.5
- Visual Studio Code
- Chrome
- JavaScript
- CSS

## 개발 환경 실행 

```bash
$ git clone git@github.com:dlog6649/labeling-tool.git
$ cd labeling-tool
$ npm i
$ npm start
```
